🧠 MemoryGlasses — Technical Deep Dive

"Imagine your grandmother forgetting who you are… but with our glasses, she smiles and says your name again."

🏗 System Architecture

MemoryGlasses is a wearable AI assistive device designed for Alzheimer's patients. The system integrates a miniature camera, microphone, Bluetooth Low Energy (BLE) module, and a companion smartphone app into lightweight smart glasses. An onboard AI pipeline identifies faces, recalls stored memories, and delivers real-time audio feedback directly to the wearer.

⚙ Core Pipeline

  • Camera captures a continuous video feed from the wearer's perspective
  • Face recognition model identifies people from a pre-built personal database
  • Memory retrieval engine pulls stored context: name, relationship, last interaction
  • Text-to-speech module delivers a soft audio cue via earpiece ("That's your daughter, Priya")
  • BLE module syncs identified faces & events to the caregiver's companion app in real time
  • Cloud backend stores interaction logs, summaries, and emotion tags for caregiver review

🤖 AI & Technology Stack

  • Face recognition: DeepFace / FaceNet model for real-time identification
  • On-device inference: TensorFlow Lite for low-latency embedded processing
  • NLP summarisation: GPT-based API for generating daily interaction summaries
  • Emotion tagging: sentiment analysis on conversation audio snippets
  • BLE communication: Bluetooth 5.0 for caregiver app sync
  • Companion app: React Native cross-platform mobile interface
  • Cloud: Firebase for real-time database and push notifications

💡 Why It Stands Out

  • Real social impact — directly addresses dignity, safety, and independence for Alzheimer's patients
  • Unique end-to-end approach combining AI, wearables, and caregiver integration
  • Feasible prototype path — basic camera + mic + BLE + phone app is enough as proof-of-concept
  • Scalable vision with cloud storage, summarisation, and emotion tagging extensions
  • Strong storytelling and emotional pitch for ideathon judges

🚧 Prototype Roadmap

  • Phase 1 — Basic prototype: camera module + face recognition on Raspberry Pi Zero
  • Phase 2 — Audio feedback via bone conduction speaker integrated into glasses frame
  • Phase 3 — BLE sync to companion Android/iOS app for caregiver alerts
  • Phase 4 — Cloud backend for memory storage, logs, and AI summarisation
  • Phase 5 — Miniaturisation and form factor refinement for real-world wearability

🎯 Applications & Impact

  • 55 million+ people worldwide live with dementia — growing 10 million per year
  • Reduces caregiver burden by providing real-time alerts and daily summaries
  • Preserves patient dignity by enabling independent social interactions
  • Extensible to other memory disorders, elderly care, and cognitive rehabilitation
